KAVO — Local Preview

How to run locally:

1. Install dependencies:

```powershell
npm install
```

2. Start a static server (local port 5000):

```powershell
npm start
```

Then open http://localhost:5000 in your browser.

Files of interest:
- `index.html` — main page
- `css/styles.css` — styles
- `js/main.js` — interactions
